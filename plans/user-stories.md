# User Stories

## Adding food / Drinks

### As a User I want to be able to add food items via scanning the barcode for breakfast, lunch, dinner and snacks.

### As a User I want to be able to add food items manually for breakfast, lunch, dinner and snacks.

### As a User I want to be able to create custom meals with nutrition information.

### As a User I want to be able to select previously created custom meals and add them to my daily log.

### As a User I want to be able to enter water intake amounts

### As a User I want to be able to enter caffeine intake amounts. 

## Adding exercise / activity

### As a User I want to be able to add an exercise and choose from a list of exercises, enter the calories burned and the time in HR zones. 

## Dashboard

### As a user I want to be able to view my dashboard showing the current week trends. Such as Total calories consumed / target for the week. Exercise details for the week.
